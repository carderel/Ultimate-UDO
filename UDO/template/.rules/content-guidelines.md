# Content Guidelines

Standards for written content produced by this project.

---

## Voice & Tone

- **Professional** but approachable
- **Clear** over clever
- **Confident** but not arrogant
- **Helpful** and solution-oriented

---

## Structure

### Documents
- Lead with the key point
- Use headings to organize
- Keep paragraphs focused (one idea each)
- Use lists for multiple items
- Include summary/conclusion

### Communications
- State purpose upfront
- Be concise
- Include clear call-to-action
- Proofread before sending

---

## Formatting

- Use consistent heading hierarchy
- Bold for emphasis (sparingly)
- Use bullet points for lists
- Include whitespace for readability
- Tables for comparative information

---

## Language

- Use active voice
- Avoid jargon unless audience expects it
- Define acronyms on first use
- Be specific, not vague
- Avoid filler words

---

## Accuracy

- Verify facts before including
- Cite sources when appropriate
- Date-stamp time-sensitive content
- Flag uncertainty explicitly
- Update outdated content

---

## Accessibility

- Use descriptive link text
- Provide alt text for images
- Use sufficient color contrast
- Don't rely on color alone
- Structure content logically

---

## Review Checklist

Before finalizing content:
- [ ] Spell-checked
- [ ] Grammar reviewed
- [ ] Facts verified
- [ ] Links tested
- [ ] Formatting consistent
- [ ] Audience appropriate
